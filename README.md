# Weather App In Flutter

An app which will provies you weather data according to city.

# Features

* Real Time Weather Data  🌄
* Data to display 
  * Temperature 🌡️
  * Time ⏱️
  * Wind speed 🌬️
  * Smoke level 💨
  * Humidity level 💧
* Search for city 🏢
* Attractive user interface ⚡
